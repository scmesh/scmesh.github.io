# 📡 SC Mesh - Santa Catarina Meshtastic Network

O **SC Mesh** é o repositório central da documentação, guias e recursos para a rede mesh descentralizada baseada em **Meshtastic** que estamos construindo no estado de Santa Catarina.

Este projeto busca ser uma "documentação viva" para entusiastas, radioamadores, preppers e qualquer cidadão interessado em comunicação resiliente e off-grid.

---

## 🏔️ O que é o SC Mesh?

É uma rede de malha (mesh) que utiliza a tecnologia **LoRa (915MHz)** para permitir a troca de mensagens, localização GPS e telemetria sem a necessidade de internet, torres de celular ou infraestrutura centralizada. 

Em um estado com geografia desafiadora e histórico de eventos climáticos como Santa Catarina, o **SC Mesh** atua como uma camada extra de segurança e resiliência para a comunidade.

## 🛠️ Sobre este Repositório

Este site foi construído utilizando **Material for MkDocs**. O objetivo é centralizar:
* 📖 **Documentação:** Guias de configuração passo a passo.
* 🛰️ **Hardware:** Recomendações de rádios, antenas e sistemas de energia solar.
* 🗺️ **Infraestrutura:** Mapeamento de repetidores e pontos estratégicos em SC.
* 🚨 **Emergência:** Protocolos para uso da rede em situações de crise.
