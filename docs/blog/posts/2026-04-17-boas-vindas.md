# Bem-vindos ao Blog do SC Mesh

Este é o primeiro post do nosso novo blog! Aqui compartilharemos o progresso do desenvolvimento, novos hardwares suportados e notícias da comunidade.

<!-- more -->

O SC Mesh é uma iniciativa para criar uma rede mesh resiliente e aberta em Santa Catarina. Com este blog, pretendemos manter todos informados sobre:

* Novas implementações de infraestrutura.
* Tutoriais de hardware.
* Encontros da comunidade.

Fiquem ligados para mais novidades!
