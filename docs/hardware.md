### 🛰️ Antenas
A antena que vem com o rádio geralmente é básica. Para melhor performance:
* **Urbano:** Antenas de 3dBi a 5dBi para penetração em obstáculos.
* **Rural/Morro:** Antenas Colineares de 8dBi+ para longo alcance.

### 🔋 Cases e Energia
Para nós externos, use caixas herméticas com certificação **IP66**. Para sistemas solares, recomendamos painéis de pelo menos 5W para garantir operação em dias nublados.
