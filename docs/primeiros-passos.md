# Primeiros Passos

Aqui você aprenderá o básico para entrar na rede SC Mesh.

### 🛠️ O que eu preciso?
Para começar, você precisará de um rádio compatível com Meshtastic operando na frequência de **915MHz** (padrão brasileiro). Recomendamos as seguintes marcas:  
- **LILYGO**  
- **Heltec** (Econômico, ótimo para quem está iniciando).  
- **RAKwireless** (Melhor eficiência energética para repetidores solares).  
- **Seeed Studio** (Ótimos dispositivos, tanto placas para construir seu nó como nós já prontos para usar)  
- **Elecrow** (Nós já prontos para usar)  


### Configuração Inicial
1. **Baixe o App:** Disponível para [Android](https://play.google.com/store/apps/details?id=com.geeksville.mesh) e [iOS](https://apps.apple.com/us/app/meshtastic/id1586432531). 
2. **Flashe o Firmware:** Use o [Meshtastic Flasher](https://flasher.meshtastic.org/) via navegador para instalar a versão mais recente no seu hardware. 
3. **Pareamento:** Conecte via Bluetooth pelo aplicativo. 

### 📡 Canais e Frequências (Padrão SC)
Para garantir que todos se comuniquem, usamos os seguintes parâmetros:  
- **Região:** `ANZ` (915MHz).  
- **Canal Primário:** `Long Fast` (LongFast).  
- **Preset:** `LongFast`.
